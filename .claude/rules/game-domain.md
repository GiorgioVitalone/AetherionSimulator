# Game Domain Rules

## Factions (6)
1. **Verdant Glades** — Nature/growth, Green
2. **Radiant Bastions** — Light/order, White
3. **Onyx Abyss** — Dark/necromancy, Black
4. **Crimson Wastes** — Fire/war, Red
5. **Sapphire Isles** — Water/knowledge, Blue
6. **Amethyst Expanse** — Illusion/change, Purple

## Resources (Dual)
- **Mana** — Magic resource
- **Energy** — Tech resource (sometimes called "Tech" in legacy text)
- **Flexible** — Player chooses Mana or Energy
- Resource Deck: exactly 15 cards, separate from Main Deck

## Card Types
- **Hero** (H) — 1 per deck, has Life Points, transformation mechanic
- **Character** (C) — Deployed to zones, HP/ATK/ARM stats
- **Spell** (S) — One-time effect, discarded
- **Equipment** (E) — Attaches to Character
- **Transformed** (T) — Hero after transformation
- **Resource** (R) — Mana/Energy card

## Battlefield Zones (per player)
- **Reserve** (2 slots) — safe, can generate resources
- **Frontline** (3 slots) — primary combat
- **High Ground** (2 slots) — can target any opposing zone except Reserve

## Key Keywords
- Deploy, Trigger, Counter, Flash, Aura, Last Breath, Ultimate
- Haste, Rush X, Sniper, Elite, Flying, Defender, Stealth, Recycle X, Swift, Volatile

## Deck Rules
- Main Deck: 40-60 cards, max 3 copies per non-Hero
- Resource Deck: exactly 15 cards
- Hero determines allowed alignments and resource types
